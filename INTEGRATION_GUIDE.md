  - Redis Cloud: https://redis.com/try-free/

#### 2. **AWS S3** (For Whiteboard Image Upload)

- **Purpose**: Cloud image hosting & optimization
- **URL needed**: Cloudinary API key
- **Setup**: https://cloudinary.com/

- **Purpose**: Send email invites, notifications

## 📚 API Endpoints

### Authentication
- `POST /api/auth/register` - Create account
- `POST /api/auth/login` - Login user
- `POST /api/auth/refresh` - Refresh token
- `POST /api/auth/logout` - Logout

### Rooms
- `GET /api/rooms` - List user's rooms
- `POST /api/rooms` - Create new room
- `GET /api/rooms/:roomId` - Get room details
- `DELETE /api/rooms/:roomId` - Delete room
- `GET /api/rooms/slug/:slug` - Join room by slug
- `POST /api/rooms/:roomId/join` - Join existing room

### Room Content
- `GET /api/rooms/:roomId/state` - Get whiteboard/code state
- `PATCH /api/rooms/:roomId/state` - Update content

### Users
- `GET /api/users/:userId` - Get user profile
- `GET /api/users/:userId/rooms` - User's rooms

## 🔄 Real-Time Socket Events

### Whiteboard Events
- `whiteboard:draw` - Pencil strokes
- `whiteboard:shape` - Rectangle, circle, line
- `whiteboard:clear` - Clear canvas
- `whiteboard:undo` - Undo action
- `whiteboard:redo` - Redo action

### Code Editor Events
- `editor:change` - Code content change
- `editor:cursor` - Cursor position
- `editor:selection` - Text selection

### Presence Events
- `presence:join` - User joined
- `presence:leave` - User left
- `presence:cursor` - Cursor movement

### Video Chat Events
- `webrtc:offer` - WebRTC offer
- `webrtc:answer` - WebRTC answer
- `webrtc:ice-candidate` - ICE candidate

## 🛠️ How to Add Future Features

### Example: Adding Redis for Scaling
1. Set `REDIS_URL` environment variable
2. Install `@socket.io/redis-adapter`
3. Update `server/socket.ts` with Redis adapter
4. Deploy to multiple instances

### Example: Adding Image Upload to Whiteboard
1. Set `AWS_S3_BUCKET`, `AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY`
2. Create image upload endpoint
3. Update whiteboard component with upload UI
4. Store S3 URLs in room state
